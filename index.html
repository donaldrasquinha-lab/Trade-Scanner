Complete feature guide · Upstox V2/V3 API · NSE/BSE · Options · Live Charts
🔑 First-Time Setup
1. Log in to your Upstox account → Developer Console → copy your daily Access Token
2. Paste it in the Upstox Token box (top of any sidebar) and click Save Token
3. Token expires daily at midnight — paste a fresh one each trading day before market open
4. Token is held in browser memory only — never stored or transmitted elsewhere
⚠ Without a token: Index Live charts and screener candles still work. Options Desk, OI Chain, MTF OI analysis, Sector Correlation, and Market Breadth require a valid token.
⬡ Stock Screener
Scans hundreds to thousands of instruments using 120 days of daily candles and scores each one with technical indicators + Bulkowski chart patterns. Results populate live as each stock completes.
Exchange options:
NIFTY 50 Stocks
All 50 index constituents. Fastest — ~4 min at default delay. Best for daily morning scan.
NSE Equity (~2,000)
All NSE-listed equities. Takes 20–30 min. Use Symbol filter to narrow.
BSE Equity (~5,500)
Full BSE universe. Longest scan. Best run overnight or with high Max stocks limit.
NSE F&O (~200)
Futures & options eligible stocks. Good liquid mid/large-cap universe.
NIFTY50 Weekly OI Chain
ATM ±5 strikes. Fetches 5m intraday candles per strike + full pattern analysis. 11 rows.
BANKNIFTY / FINNIFTY OI
Same as above for Bank and Fin Nifty monthly chains.
Columns: Sector · LTP · Chg% · OI · OI Chg · VWAP · SMA20 · SMA50 · Slope% · ROC5% · ROC20% · Patterns · ST Score · ST Verdict · LT Score · LT Verdict
Verdicts: Strong Bull / Bull · Neutral · Bear / Strong Bear — based on 10 weighted conditions per stock
Tips: Sort by ST Score descending for top bullish picks. Use Group by Sector to see which sectors are outperforming. Export to CSV for further analysis.

⚡ MTF Signal Engine
Analyses 3 Higher Timeframe layers to generate Buy / Sell / Wait signals for 1m, 3m, 5m execution — with OI and ADX incorporated.
HTF Layer weights: 4H = 40% (macro trend) · 30m = 35% (swing confirm) · 15m = 25% (entry window)
Each HTF card scores 10 conditions: EMA9/21 cross · VWAP position · RSI zone · RSI extreme fade · SMA20/50 alignment · Price vs SMA20 · Slope · ROC momentum · Bulkowski patterns · +DI/-DI
ADX row in each card: ▲ Bull Trend (ADX≥25, +DI>-DI) · ▼ Bear Trend · ◆ Weak Trend · — Ranging
Execution signal confidence adjustments:
ADX ≥30 aligned
+8% confidence — strong trend confirmed
ADX opposes direction
−10% confidence — trend fighting the signal
PCR confirms
+7% confidence — options market agrees
OI buildup aligns
+5% confidence — smart money positioning
Levels (ATR-based): Entry · Stop Loss · Target 1 (1.5× ATR) · Target 2 (2.5× ATR) · R:R ratio. ATR multiplier scales with TF: 1m=0.5×, 3m=0.8×, 5m=1.2×
Alert: Browser notification fires when confidence ≥65% and signal changes. Click Allow when prompted.
◎ Options Desk
Live options analytics: Greeks · OI Wall · IVR/IVP · PCR trend. Select index + expiry → click ▶ Load Options Data.
Expiry dates auto-populate using correct 2025+ NSE/BSE schedule: NIFTY50 = weekly Tuesday · BANKNIFTY/FINNIFTY = monthly last Tuesday · SENSEX = weekly Thursday.
Delta
CE/PE: how much option moves per ₹1 in index. ATM ≈ 0.5. Deep ITM → 1.0.
Gamma
Rate of change of delta. Highest at ATM near expiry — gamma risk spikes last 2 days.
Theta
Daily time decay in ₹. Negative for buyers, positive for sellers. Accelerates near expiry.
Vega
₹ change per 1% IV move. Buy when IVR low (cheap vega), sell when IVR high.
OI Wall Chart: CE OI (red, left = resistance) vs PE OI (green, right = support). Max CE OI strike = strong resistance. Max PE OI strike = strong support. These are the key levels to watch.
IVR Guide: IVR <30 = cheap premiums → buy options · IVR 30–70 = normal · IVR >70 = expensive → sell options
PCR: Put-Call Ratio. PCR >1.2 = more puts = bullish. PCR <0.8 = more calls = bearish. Sparkline shows trend over last 20 loads.

⚠ Important Notes
• Token expires daily — paste a fresh Upstox token each morning before 9:15am
• Not financial advice — all signals are based on technical indicators only. Always use a stop-loss. Past patterns do not guarantee future performance.
• No data stored — everything runs in your browser. Closing the tab clears all data.
• Rate limits (HTTP 429) — if scans fail, increase the Delay (ms) setting in Screener sidebar.
• IVR is approximate — uses a fixed 10–45% IV range proxy. For precise IVR, 52-week historical IV data is needed.
• OI Chain ATM ±5 strikes — fetches live spot price to find ATM, then ±5 × strike spacing. If spot fetch fails, uses chain midpoint.
• Nifty 50 constituent list — current as of June 2026 including BEL, ETERNAL, INDIGO, JIOFIN, MAXHEALTH additions. If a stock is replaced, update the list in source.
• Expiry schedule (post Sep 2025): NIFTY50 = weekly Tuesday · BANKNIFTY = monthly last Tuesday (weekly discontinued Nov 2024) · SENSEX = weekly Thursday.
